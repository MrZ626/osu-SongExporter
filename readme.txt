这是一个osu歌曲导出器！

本质功能：将工程文件夹内(对，工程文件夹内，不能指定，没办法)的所有文件夹里最大的一个音频文件(wav/mp3/ogg)提取出，重命名成文件夹名字里第一个空格之后的内容(osu的格式，一串数字加一个空格再加完整歌曲名)

注意：会自动忽略名字完全一致的歌曲，比如同一个曲有两个谱面，就会有两个id不一样但是歌名部分一样的文件夹，会取其中一个生成，忽略其他的(具体哪个不晓得)

This is an OSU song exporter!

Essential function: put the project folder(yes, in the project folder,can not be specified, no way) The largest audio file (WAV / MP3 / Ogg) in all the folders is extracted, rename to the content after the first space in the folder name (in OSU format, a string of numbers plus a space, plus the full song name)

Note: songs with exactly the same name will be automatically ignored. For example, there is a song has two beatmaps, so there will be two folders with different ID but the same song title. Only one of them will be choosed to extract and the others will be ignored.
